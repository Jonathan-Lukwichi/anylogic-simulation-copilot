# Keeping the knowledge base current

Refresh cadence:
- On every AnyLogic release: re-check anylogic.help block/RL/Alpyne pages (these are fetch-live).
- Quarterly: re-pull the download sources, update dates in SOURCES.md.
- When you find a strong new open-access review/tutorial: add it to the right category + SOURCES.md.

Rules:
- Only legitimate/free/open-access sources. No pirated copies.
- Prefer clean text/Markdown for downloaded files (best for retrieval).
- Record title, URL, licence, and date pulled in SOURCES.md for every file.
